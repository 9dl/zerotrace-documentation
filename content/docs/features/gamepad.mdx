---
title: 🎮 Gamepad Controls
---

ZeroTrace can **emulate a USB gamepad** for advanced HID automation and testing.
- 🕹️ **Press and hold any gamepad button** (A, B, X, Y, L1/R1, L2/R2, Start, Select, Home)
- 🎯 **Precise analog stick control** with X/Y values (`-127` to `127`)
- ➡️ **D-Pad (Hat) support** for all 8 directions and neutral position
- ⚡ **Rapid-fire button tapping** for automation or testing
- 🔀 **Combine with keyboard/mouse scripts** for multi-device payloads
- 🛠️ **Ideal for game automation, testing, or bypassing input restrictions**
