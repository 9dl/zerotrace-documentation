---
title: 📶 WiFi Connectivity
---

ZeroTrace can **connect to WiFi networks** for wireless control and automation—no external antenna required!
- 🌐 **Station mode**: Connect ZeroTrace to your existing WiFi network and control it from any device on the same network
- 📡 **Access Point (AP) mode**: ZeroTrace creates its own WiFi hotspot, allowing direct connections for setup or offline use
- 🛠️ **Configure network credentials** easily via scripts or web interface
- 🔄 **Switch between Station and AP modes** on the fly
- 📲 **Remote payload deployment**: Control and trigger scripts over WiFi
- 🔒 **Secure communication** with optional password protection for AP mode
- 🚫 **No need for external dongles or antennas**—fully integrated wireless functionality
