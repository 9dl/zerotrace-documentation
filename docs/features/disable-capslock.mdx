---
title: ⌨️ Disable Capslock Feature
---

Ensure **Caps Lock is turned off** when executing scripts to prevent unintended uppercase input that could break case-sensitive commands.
- ✅ **Prevents script execution errors** due to unintended capitalization
- ⚙️ **Automatically disables Caps Lock** when active
- 🔧 **Configurable** in the **Settings** section
