---
title: 🖥️ Display Support
---

ZeroTrace features **integrated display support** for improved usability and feedback.

- 📺 View payload execution in real time (teleprompting like functionality)
- 🔔 Receive visual alerts for errors, completion, or user actions
- ⚡ Streamline setup, diagnostics, and deployment without external tools

> 💡 **Tip:** Display availability and features may vary between ZeroTrace models.
> Check the [store](https://www.zerotrace.pw/store) to see which devices specifically include display support.
