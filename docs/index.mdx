# Welcome to ZeroTrace

**ZeroTrace** is a next-generation, WiFi-enabled keystroke injection and HID automation platform built on the ESP32-S3. Engineered for precision, performance, and security, it empowers pentesters, developers, and automation enthusiasts with reliable HID automation and seamless web-based control.

<Callout type="info" title="Why ZeroTrace?">
ZeroTrace delivers advanced scripting, real-time web management, and robust automation capabilities in a compact, open-source device.
</Callout>

## Overview

ZeroTrace is designed for security testing, automation, and custom scripting. It offers:

- High-speed HID injection for Windows, macOS, Linux, Android, and iOS.
- Web-based dashboard for managing, scripting, and real-time testing.
- Script automation using a documented language and API.
- Stealth and spoofing features for advanced scenarios.
