# Installation & Setup

Get started with ZeroTrace in just three simple steps: **Prepare ‚Üí Flash ‚Üí Connect**.

---

## 1. Preparation

**You'll Need:**
- ZeroTrace device
- USB-C data cable
- Computer (Windows, macOS, or Linux)
- Chrome or Edge browser (recommended)
- Internet connection

---

## 2. Port Guide

ZeroTrace has two USB ports:

| Port | Use For           | Example           |
|------|-------------------|-------------------|
| **COM** | Updates, tools     | Firmware flashing |
| **USB** | HID payloads       | Keystroke attacks |

<Callout type="info">
    üí° <b>Tip:</b> Use <b>COM</b> for device maintenance and updates. Use <b>USB</b> for payload operations.
</Callout>

---

## 3. Flash Firmware

<Tabs items={['All Platforms']}>
    <Tab value="All Platforms">
        1. Connect ZeroTrace to your computer via the **COM** port using a USB-C data cable.
        2. Open the [ZeroTrace Web Flasher](https://dashboard.zerotrace.pw/) in Chrome.
        3. Click **Connect**, select your device, then choose **Flash Firmware**.
        4. Wait (~2 minutes) for the process to finish, then unplug and restart your device.
    </Tab>
</Tabs>

**Linux Only:**
If you see `Failed to execute 'open' on 'SerialPort'`, run:
```bash
sudo usermod -a -G dialout $USER
```
Then **log out and back in**.

---

## 4. Connect to ZeroTrace

| Setting      | Value                  |
|--------------|------------------------|
| SSID         | `[DeviceName] ZT`      |
| IP Address   | `192.168.4.1`          |
| Password     | `ZeroTrace`            |

**Steps:**
1. Power on ZeroTrace.
2. Connect to the WiFi network ending with `ZT` (password above).
3. Visit [`http://192.168.4.1`](http://192.168.4.1) in your browser.

## 5. Troubleshooting

| Issue                | Solution                                   |
|----------------------|--------------------------------------------|

| WiFi not showing     | Wait 2‚Äì3 minutes, check for WiFi conflicts |
| Flash fails          | Use a reliable cable, ensure stable power  |
| Timeout/Error        | Verify WiFi and IP, restart device         |

- For persistent WiFi issues, perform a factory reset.
