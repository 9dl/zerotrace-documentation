### Default Network Credentials

|--------------|----------------------|

### Connection Steps
1. Power on the ZeroTrace device.
2. Find and connect to the WiFi network ending with `ZT`.
3. Use the default password provided above.
4. Open [`http://192.168.4.1`](http://192.168.4.1) in your browser to access the web dashboard.
