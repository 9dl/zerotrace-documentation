# ZeroTrace Web Interface

A **browser-based control center** to manage payloads, run scripts, and configure your device wirelessly.

---

## 🌟 Core Features

### ✍️ Script Executor
![](/executor.png)
- Instantly run and test scripts or payloads in real time no reflashing needed.

---

### 📁 File Manager
![](/filemanager.png)
- Upload, edit, and organize your scripts and payload files.

---

### 🎮 Live Mode
![](/livemode.png)
- Directly control the mouse, send keystrokes, draw, or type live as a keyboard.

---

### ⚙️ Settings Panel
![](/settings.png)
- Adjust WiFi, power settings, device preferences, and more.

---

## 📈 Workflow Advantage

**Old HID Device Workflow:**

**ZeroTrace Workflow:**
