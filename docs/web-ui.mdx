# ZeroTrace Web Interface

**Browser-based control center** for managing payloads, executing scripts, and configuring device settings wirelessly.

## 🌟 Core Functionality

---

### ✍️ Script Executor
![](/executor.png)
- Instantly run scripts and test payloads in real time without reflashing or disconnecting.

---

### 📁 File Manager
![](/filemanager.png)
- Manage, upload, edit, and organize your scripts and payloads.

---

### 🎮 Live Mode
![](/livemode.png)
- Control the mouse, draw with the mouse, send keystrokes, or write live in the input box as a keyboard.

---

### ⚙️ Settings Panel
![](/settings.png)
- Configure WiFi, power options, device preferences, and more.

---

## 📈 Workflow Comparison

**Traditional HID Device Cycle:**
`Edit → Compile → Flash → Test → Repeat`

**ZeroTrace Workflow:**
`Edit → Test → Execute`
